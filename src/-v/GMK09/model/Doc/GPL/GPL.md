## 优雅地证明几何定理
> ▪ 通过将`自然语言`描述的定理转换为`GPL`证明语言，消除歧义，`描述`几何之间的关系，计算机通过建立`大量`，`随机`且`符合题设`的几何模型，在`数值`上验证定理正确性


### ▪勾股定理 示例

#### 自然语言
```\n勾股定理指直角三角形的两条直角边长（古称勾长、股长）的平方和等于斜边长（古称弦长）的平方。
```

#### GPL程序
```\n@ Triangle t is Right
? Is.Equal( (#(t.a-t.b))^2 + (#(t.a-t.c))^2 , (#(t.b-t.c))^2 )
```

### ▪三角形的中位线 示例

#### 自然语言
```\n定义：连接三角形两边中点的线段叫做三角形的中位线。
定理：三角形的中位线平行于三角形的第三边，并且等于第三边的二分之一。
```

#### GPL程序
```\nξ验证三角形的中位线等于底边的一半
$ Point A
$ Point B
$ Point C
! Point.is3PCollinear(A,B,C)
@ Point D is mid of A,B
@ Point E is mid of A,C
? Is.Equal( #(D-E)*2 , #(B-C) )
```
