# Circle

> 圆，全称圆形，是指在一个平面内，一动点以一定点为中心，以一定长度为距离旋转一周所形成的封闭曲线，是平面上到定点的距离等于定长的所有点的集合，其定点称为圆心，定长称为半径[1]。圆又是一个“正无限多边形”，当多边形的边数越多时，其形状、周长、面积就都越接近于圆，所以世界上没有真正的圆，只有一种概念性的图形。[📡 Circle](https://baike.sogou.com/m/fullLemma?key=圆)

### ▪在声明时使用
使用声明符号`$`作为行头标识，`Circle`随其后，最后标注对象`标签`，不要忘记用空格隔开，这样一个圆周就被创建完成了
下面是一个示范，在平面中创建圆c

```\n$ Circle c
```

### ▪在约束时使用

遵循约束的基本格式

```\n@ Circle c is method of factors
```

如果想创建过三点的圆，像这样操作

```\n@ Circle c is newFrom3P of A,B,C
```
